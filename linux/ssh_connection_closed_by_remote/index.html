SSH remote failed QA

SSH connection closed

Source: https://linux.cn/article-4226-1.html

Status: SSH: Connection closed by

Q1: When I tried to connect to a remote server through SSH. It failed and pop the following message on my console “Connection close by 127.0.0.1”? A: Something goes wrong with your SSH key on the remote server reinstall SSH Host keys by: sudo dpkg-reconfigure openssh-server

Step 3. Re-create SSH host keys

# remove the old keys
$ sudo rm -r /etc/ssh/ssh*key
$ sudo dpkg-reconfigure openssh-server
